# 简中互联网大事件日历

> 🗓️ 记录、分析和可视化简中互联网的热点事件与历史大事

## 📖 项目简介

**简中互联网大事件日历**是一个集事件记录、数据分析、可视化展示于一体的综合性平台。项目旨在系统性地收集和整理简体中文互联网中的热点事件、流行文化和历史大事，为用户提供直观的时间线浏览、智能搜索和深度分析功能。

## 🎯 核心价值

- **📚 文化存档** - 系统性记录互联网文化演变历程
- **🔍 智能检索** - 多维度搜索和筛选历史事件
- **📊 数据可视化** - 直观展示事件热度和趋势变化
- **🤖 智能分析** - 基于AI的事件分类和情感分析
- **🌐 多源采集** - 整合各大平台热点数据

## ✨ 特色功能

### 🏛️ 万神殿
记录历史名人，永垂不朽的人物事迹档案

### 📅 同年今日
回顾历史上的今天发生的重大事件

### 🎭 梗图生成
基于热点事件自动生成抽象梗图和表情包

### 📋 数据管理
完整的事件CRUD操作，支持批量导入导出

### 🌐 网络采集
自动化采集各大社交平台热点内容

## 🏗️ 技术架构

### 后端技术栈
- **语言**: Python 3.8+
- **Web框架**: FastAPI
- **ORM**: SQLAlchemy 2.0
- **数据库**: MySQL 9.5 + SQLite (开发备用)
- **异步处理**: asyncio

### 前端技术栈
- **桌面端**: Tkinter/ttk + PyQt (双界面支持)
- **Web端**: React (规划中)
- **移动端**: Flutter (规划中)

### 数据架构
数据采集 → 数据清洗 → 数据分析 → 数据可视化
↓ ↓ ↓ ↓
多源API 标准化处理 机器学习模型 图表引擎
